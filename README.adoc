== Compile and install

You will need Java 17 installed.
If you do not have it, get it from sdkman.io.

.Init project
[source,bash]
----
./mvnw verify
cp app/target/*unojar*.jar $HOME/.local/apps/
# execute
java -jar $HOME/.local/apps/
----
